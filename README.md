# PaperTrader

Welcome to **PaperTrader**, a full‑stack TypeScript/React Native project
designed for simulated stock trading challenges.

## Features

-   Real‑time stock data visualization.
-   Customizable trading challenges.
-   Modular architecture with components, hooks, utilities, and state
    management.
-   TypeScript‑based codebase.
-   Cross‑platform mobile support.

## Getting Started

### Prerequisites

-   Node 14+
-   npm or yarn
-   Expo CLI or React Native CLI

### Installation

    git clone https://github.com/tonderaiitai8-png/tekbro.git
    cd tekbro
    npm install

### Running

    npm run start
    npm run android
    npm run ios

## Project Structure

    /app
    /components
    /constants
    /contexts
    /hooks
    /store
    /utils
    types.ts
    package.json

## How to Play

1.  Launch the app.
2.  Choose a challenge.
3.  Execute trades with simulated capital.
4.  Track performance and progress.
